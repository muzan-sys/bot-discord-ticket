# Discord Ticket Bot

A ticket management system for Discord servers that helps organize user requests and support tickets efficiently.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/muzan-sys/bot-discord-ticket.git
   ```
2. Navigate into the project directory:
   ```bash
   cd bot-discord-ticket
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage

1. Configure your bot token and other settings in the `config.json` file.
2. Start the bot:
   ```bash
   node index.js
   ```
3. Users can create a ticket by using the command `!ticket` in your server.

## Features
- Easy ticket creation through simple commands.
- Automatic ticket assignment to support staff.
- Ticket closure with a single command.
- Logging of ticket interactions for transparency.
- Customizable settings for ticket channels and roles.
